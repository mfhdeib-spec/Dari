# Clothing product images

Product images for the clothing section. The site references these by filename (e.g. `/clothing/09-black-thobe.png`).

## Replacing 09-black-thobe.png (Black Thobe card)

To use the emerald green embroidered abaya image for the Black Thobe product card:

1. Locate the image file (e.g. from Cursor attachments: `Google_Gemini_Generated_Image__1___1_-aaa72fec-897f-4e05-9f9f-82ff6935e4ff.png`).
2. Copy or save it into this folder as **`09-black-thobe.png`**, overwriting the existing file.
3. Refresh the site; the product card will show the new image.
